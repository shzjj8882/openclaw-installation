# 功能概览

OpenClaw 安装助手是一款基于 **Electron** 的桌面应用，为 OpenClaw 提供图形化安装与配置体验。

## 核心功能

### 1. 环境检测与安装

应用启动时自动检测运行环境：

| 检测项 | macOS | Windows | Linux |
|--------|-------|---------|-------|
| Node.js | ✅ | ✅ | ✅ |
| Homebrew | ✅（仅 macOS） | 跳过 | ✅ |
| OpenClaw | ✅ | ✅ | ✅ |

支持一键安装各项依赖，无需记忆复杂命令。

### 2. 安装流程

分步引导用户完成依赖安装：

1. **Node.js**：提供官方下载或内置安装
2. **Homebrew**（macOS）：提供安装脚本与文档
3. **OpenClaw**：支持图形化安装

每步完成后可点击「下一步」继续，环境检测通过后自动进入配置流程。

### 3. 配置流程

与 openclaw 配置流程保持一致，从应用内动态获取配置选项：

- **模型与认证**：选择模型提供商（Ollama、OpenAI、自定义等），填写 API Key 等
- **频道配置**：选择飞书、钉钉等频道，按需填写 Token、策略等
- **Skills 配置**：选择要安装的 Skills，支持手动添加
- **应用配置**：一键应用，无需打开终端

### 4. 安装后操作

环境就绪后提供：

- **启动 / 重新启动**：根据 Gateway 状态显示对应按钮
- **更改配置**：进入内置终端进行配置
- **查看日志**：应用内实时展示 Gateway 日志
- **后台启动**：可选后台运行 Gateway

### 5. 内置终端

- 配置终端：进行交互式配置
- 日志终端：只读模式展示实时日志
- 支持 ANSI 颜色、滚动、窗口缩放
