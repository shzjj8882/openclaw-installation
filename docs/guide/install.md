# 安装流程

安装流程按顺序引导用户完成 Node.js、Homebrew（macOS）、OpenClaw 的安装。

## 流程步骤

### 1. Node.js

应用会自动检测是否已安装 Node.js。

**安装方式**：

- 点击「安装 Node.js」进行一键安装
- 或通过「打开官网」跳转 [nodejs.org](https://nodejs.org/) 手动安装

安装过程中会显示进度条与状态提示。

### 2. Homebrew（仅 macOS）

::: tip 平台说明
Windows 不检测 Homebrew，安装流程中会跳过此步骤。
:::

应用会自动检测是否已安装 Homebrew。

**安装方式**：

- 应用内提供 Homebrew 官方安装脚本
- 或通过「打开官网」跳转 [brew.sh](https://brew.sh/) 手动安装

### 3. OpenClaw

应用会自动检测是否已安装 OpenClaw。

**安装方式**：

- 点击「安装 OpenClaw」进行一键安装
- 安装过程中会显示进度与提示（如「即将弹出授权窗口，请输入密码」）

## 自动跳转

当检测到 **Node.js**、**OpenClaw** 均已安装，且 **Homebrew**（如需）已安装时，应用会自动进入配置流程，无需手动点击「下一步」。
